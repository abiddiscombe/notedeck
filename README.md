NoteDeck is a free sticky notes board that works offline. Your notes are saved in your browser (using [IndexedDB](https://developer.mozilla.org/en-US/docs/Web/API/IndexedDB_API)) and never leave your device.

## Features

- Light & Dark Modes.
- File-Based Backup & Restore Capability
- Installable as a Progressive Web App for offline / native use.
- Provides resizable "sticky notes" with support for multiple themes.
